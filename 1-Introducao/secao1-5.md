<h1 align="center">Escopo</h1>

<p align="center"><i>Esta seção não é normativa</i></p>

<p align="justify">
  Esta especificação se limita a fornecer uma linguagem de marcação em nível semântico e APIs de script associadas também em nível semântico para a criação de páginas acessíveis na web, que variam de documentos estáticos a aplicativos dinâmicos.
</p>

<p align="justify">
  O escopo desta especificação não inclui o fornecimento de mecanismos para personalização da apresentação específica para cada mídia (embora as regras de renderização padrão para navegadores da web estejam incluídas no final desta especificação, e vários mecanismos para integração com CSS sejam fornecidos como parte da linguagem).
</p>

<p align="justify">
  O escopo desta especificação não é descrever um sistema operacional completo.
  Em particular, softwares de configuração de hardware, ferramentas de manipulação de imagens e aplicativos que os usuários normalmente utilizariam em estações de trabalho de alto desempenho estão fora do escopo.
  Em termos de aplicativos, esta especificação se destina especificamente a aplicativos que os usuários utilizariam ocasionalmente ou regularmente, mas em locais distintos, com baixos requisitos de CPU.
  Exemplos desses aplicativos incluem sistemas de compras online, sistemas de busca, jogos (especialmente jogos online multiplayer), listas telefônicas ou agendas de endereços públicos, softwares de comunicação (clientes de e-mail, clientes mensagens instantâneas, softwares de discussão), softwares de edição de documentos, etc.
</p>
