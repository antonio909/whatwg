# Contexto

*Esta seção é informativa.*

HTML é a principal linguagem de marcação da World Wide Web. Originalmente,
o HTML foi projetado principalmente como uma linguagem para descrever 
semanticamente documentos científicos. Seu design geral, entretanto, permitiu 
que ele fosse adaptado, ao longo dos anos seguintes, para descrever diversos
outros tipos de documentos e até mesmo aplicações.
