# Isso é HTML5?

*Esta seção é informativa.*

Resumindo: Sim.

De forma mais detalhada: o termo 'HTML5' é amplamente utilizado como um jargão
(ou buzzword) para se referir às tecnologias web modernas, muitas das quais
(embora de forma alguma todas) são desenvolvidas no **WHATWG**. Este documento
é um deles; outros estão disponíveis na visão geral de normas da WHATWG.
