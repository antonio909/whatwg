# Escopo

*Esta seção é informativa.*

Esta especificação limita-se a fornecer uma linguagem de marcação de nível
semântico e APIs de script de nível semântico associadas para a criação de
páginas acessíveis na web, abrangendo desde documentos estáticos até
aplicações dinâmicas.

O escopo desta especificação não inclui o fornecimento de mecanismos para a
customização da apresentação específica de mídias (embora regras de
renderização padrão para navegadores web estejam incluídas ao final desta
especificação, e diversos mecanismos para integração com CSS sejam fornecidos
como parte da linguagem).

O escopo desta especificação não é descrever um sistema operacional inteiro. Em particular,
softwares de configuração de hardware, ferramentas de manipulação de imagem e aplicações
que se esperaria que usuários utilizassem em estações de trabalho de alto desempenho
diariamente estão fora do escopo.

Em termos de aplicações, esta especificação é voltada especificamente para aplicações que
se esperaria que fossem utilizadas por usuários ocasionalmente, ou regularmente mas de
locais distintos, com baixos requisitos de CPU. Exemplos de tais aplicações incluem sistemas
de compras online, sistemas de busca, jogos (especialmente jogos online multiplayer), listas
telefôncias públicas ou agendas de contatos, softwares de comunicação (clientes de e-mail,
clientes de mensagens instantâneas, softwares de discussão), softwares de edição de
documentos, etc.
